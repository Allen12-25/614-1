個人介紹作品

學習過程

  • 剛開始接觸 HTML 和 CSS 的時候其實滿不習慣的，有些語法常常記不起來。
  • 上課時有照著老師的簡報一步步做，回家也會再練習。
  • 後來試著加上一點 JavaScript 做互動，像是按鈕切換效果。
  • 做 RWD 響應式的那份作業花了比較久時間，但完成後覺得很有成就感。
  • 整個過程中有遇到一些困難，不過查資料和問同學、甚至問 ChatGPT，都有幫上忙。在這門課也有學到如何用github繳交作業

心得與反思

  • 這門課讓我第一次真的動手做出網頁，從排版到配色都有接觸到。
  • 雖然一開始很不熟，但後來慢慢熟悉後就越來越有興趣。
  • 其中我覺得最有挑戰的是讓版面在手機上看起來也漂亮（RWD），但也覺得這是很實用的技能。
  • 希望未來可以學更多像是網頁動畫或資料互動的功能，讓作品更完整。
